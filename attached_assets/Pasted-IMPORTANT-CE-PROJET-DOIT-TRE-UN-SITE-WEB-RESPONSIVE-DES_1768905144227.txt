IMPORTANT : CE PROJET DOIT ÊTRE UN SITE WEB RESPONSIVE (DESKTOP + MOBILE), PAS UNE APPLICATION MOBILE.
NE PAS UTILISER React Native, Expo, navigation par tabs en bas, ni le concept de “screens”.
C’est un site vitrine classique avec un header (navbar en haut), des pages web, et un footer.

OBJECTIF
Créer un site vitrine premium en FR pour une agence : “C.D.CALL – Performance & Acquisition”.
Thème : digitalisation, création de sites web, SEO, Meta Ads, tracking, génération de leads.
Style : bleu nuit/gris clair avec accent orange. Design moderne, crédible, orienté conversion.

STACK OBLIGATOIRE
- Next.js (App Router) + TypeScript
- TailwindCSS
- Composants réutilisables (Navbar, Footer, Container, Section, Card, Button, FAQAccordion, etc.)
- Optionnel : Framer Motion (animations discrètes uniquement)

ASSETS
- Le logo est cdcall.png → place-le dans /public et utilise-le dans la navbar + hero.

ROUTES / PAGES
1) /               → Accueil
2) /a-propos       → À propos
3) /services       → Services
4) /audit-contact  → Audit & Contact

EXIGENCES RESPONSIVE (très important)
- Desktop-first : rendu magnifique sur ordinateur (>= 1280px), puis responsive mobile.
- Utiliser un container global : max-w-7xl mx-auto px-6 lg:px-10
- Utiliser des grilles sur desktop :
  - Hero en 2 colonnes dès lg (texte gauche, visuel / carte droite)
  - Offres en 3 colonnes dès lg (lg:grid-cols-3)
  - Clients en 2 colonnes dès md
- Navbar :
  - liens visibles en ligne dès md
  - menu burger uniquement sur mobile
- Typo confortable :
  - titres : text-4xl lg:text-6xl
  - paragraphes : text-base lg:text-lg leading-relaxed
- Espacements : sections py-16 lg:py-24

CONTENU LONG (OBLIGATOIRE)
Chaque page doit avoir du contenu long et riche (pas des mini phrases).
Je veux :
- des paragraphes complets
- des listes détaillées
- des exemples concrets
- une méthode de travail claire
- des CTA réguliers
- une FAQ (5 à 8 questions avec réponses longues)
Le ton doit être crédible (agence sérieuse), pas “blabla marketing”.

NAVBAR (toutes pages)
- Logo + “C.D.CALL”
- Liens : Accueil / À propos / Services / Audit & Contact
- Bouton CTA : “Demander un audit”

FOOTER (toutes pages)
- Résumé court + liens + placeholders (email / WhatsApp / ville)
- Mention : “© C.D.CALL – Performance & Acquisition”
